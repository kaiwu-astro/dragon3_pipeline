# Dragon3 Pipelines Documentation

## Overview

Dragon3 Pipelines is a Python package for analyzing and visualizing N-body simulation data from Dragon3 simulations.

## Installation

```bash
# Clone the repository
git clone https://github.com/kaiwu-astro/dragon3_pipeline.git
cd dragon3_pipeline

# Install in development mode
pip install -e .
```
